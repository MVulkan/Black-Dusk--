Getting Started (Quick Test)
---------------------------
- Enable the `Black_Dusk` mod in the HOI4 launcher.
- Start a new game with a bookmarked start date (2028.1.1 is the designed start).
- Check the launcher logs (Documents/My Games/Paradox Interactive/Hearts of Iron IV/logs/error.log and error.log.old) for parse errors.
- Run the included localisation checker by opening PowerShell and running:

```powershell
powershell -ExecutionPolicy Bypass -File "mod\Black_Dusk\tools\check_localisation.ps1"
```

Known issues:
- Some event/localisation keys were missing; running the localisation checker will help identify if anything else is missing.
- The mod uses overridden `history/states/` files; ensure your province lists are correct for the base map if you change them.
